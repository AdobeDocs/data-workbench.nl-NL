# Een werkruimte delen{#share-a-workspace}

Met het menu van de werkruimte Exporteren kunt u werkruimten afdrukken en exporteren naar .png-bestanden of naar Excel.

Klik in een werkruimte op **[!UICONTROL Export]**.
